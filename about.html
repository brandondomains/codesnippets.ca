About

CodeSnippets.ca is a lightweight index for useful developer patterns.

Why this exists

GitHub Gists are great for storing reusable code, but they are not always ideal for browsing, grouping, or explaining a growing snippet library.

This Jekyll site gives each snippet a clean landing page, search-friendly metadata, topic browsing, and a direct path back to the canonical Gist.
